# Support Channels

## 🐞 Bug Reports

If you have found a bug, please create a GitHub Issue using the **Bug Report** template.

Include:

- Clear description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs
- Screenshots (if applicable)

---

## 💡 Feature Requests

Use the **Feature Request** template.

Include:

- Business problem
- Proposed solution
- Business value
- Acceptance criteria

---

## ❓ Questions

If you have development or architecture questions, use the **Question** issue template.

Examples:

- Architecture clarification
- Development workflow
- API usage
- Database design
- Best practices

---

## 📚 Documentation

Documentation improvements are always welcome.

Use the **Documentation** issue template to suggest:

- Missing documentation
- Corrections
- Better examples
- Diagrams
- Tutorials
- Runbooks
