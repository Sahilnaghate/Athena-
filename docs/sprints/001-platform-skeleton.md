# Sprint 001 — Platform Skeleton

## Goal

Establish ATHENA's production-oriented technical baseline without introducing business-domain functionality.

## Scope

- pnpm and Turborepo workspace
- Next.js web and admin applications
- FastAPI API and worker process
- PostgreSQL and Redis local dependencies
- health endpoint, configuration, logging, authentication boundary, and CI

## Explicitly out of scope

Market logic, scanners, AI features, and domain-specific business services.
